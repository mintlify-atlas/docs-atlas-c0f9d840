title: "Research intelligence reports in NEXUS"
description: "Synthesize your paper library into reports covering key consensus, research gaps, contradictions, temporal trends, and a five-metric confidence matrix."
---

The Insights feature synthesizes your entire session paper library into a structured research intelligence report. Rather than asking a free-form question, NEXUS samples representative content from across all indexed papers and prompts the language model to produce a structured JSON analysis covering areas of agreement, unresolved questions, conflicting results, and how the field has evolved over time.

## What's in an insights report

<CardGroup cols={2}>
  <Card title="Key Consensus" icon="circle-check">
    Points of agreement found across multiple papers—findings that appear consistently regardless of methodology or research group.
  </Card>
  <Card title="Research Gaps" icon="circle-question">
    Areas the surveyed literature has not addressed, under-explored directions, or open problems explicitly acknowledged by the authors.
  </Card>
  <Card title="Contradictions" icon="triangle-exclamation">
    Conflicting findings between papers—cases where different studies report incompatible results, conclusions, or interpretations.
  </Card>
  <Card title="Temporal Trends" icon="chart-line">
    How the field has evolved over time based on publication years and shifting methodological or conceptual focus across the papers.
  </Card>
</CardGroup>

## Confidence matrix

Every insights report includes a **Confidence Matrix** with five metrics, each scored 0–1 and displayed as a percentage:

| Metric | Description |
|---|---|
| `factual_accuracy` | How well the synthesized claims are grounded in the retrieved source passages |
| `source_diversity` | Breadth of distinct papers and authors contributing to the synthesis |
| `temporal_coverage` | Range of publication years represented in the sampled chunks |
| `consensus_level` | Degree of agreement across papers on core findings |
| `methodological_rigor` | Consistency and specificity of experimental approaches described |

These scores are generated by the LLM alongside the report and reflect its assessment of the available evidence—they are not computed algorithmically.

## Key themes

Below the confidence matrix, NEXUS surfaces a set of **key themes**: extracted keywords and conceptual labels that recur across the sampled passages. Themes are displayed as colored tag pills and give a fast overview of the dominant topics in your paper library.

## How insights are generated

When you open the Insights tab or click **⟳ Refresh**, NEXUS calls `GET /api/sessions/{id}/insights`, which runs the following pipeline:

1. Embed the query `"key findings methodology results contribution"` using BAAI/BGE-M3.
2. Search Qdrant for the **top 20 most relevant chunks** across all papers in the session.
3. Assemble a context block with up to 500 characters per chunk, attributed by paper title and year.
4. Prompt the LLM to return a structured JSON object with `consensus`, `gaps`, `contradictions`, `trends`, `key_themes`, and `confidence_scores` fields.
5. Parse and return the JSON to the frontend for rendering.

The full prompt structure enforces strict JSON output so that each section renders into the correct card without post-processing.

## Using the Insights tab

1. Select a session from the sidebar. The session must contain at least one indexed paper.
2. Click **⧖ Insights** in the header navigation.
3. Click **⟳ Refresh** to generate or regenerate the report.
4. Results appear in the four synthesis cards, the confidence matrix bars, and the key themes section.

Switching sessions and clicking **Refresh** re-runs the synthesis against the new session's paper library. There is no cached state—each refresh is a fresh LLM call.

<Warning>
Insights require at least one indexed paper in the active session. If the session has no papers, the endpoint returns a `400 Bad Request` with the message "No papers in session". Index your papers first via the Library tab before generating insights.
</Warning>

<Tip>
Add focused, thematically related papers to get more coherent insights. A session containing papers on a single topic (e.g., diffusion models for image synthesis) will produce sharper consensus and contradiction analysis than a heterogeneous collection covering multiple unrelated fields.
</Tip>
